#update_WebsiteAndEmail


## 介绍：

### 运行环境：

- python2.7
- pycharm


### 说一下这个爬虫的作用：

1. 获取部分需要更新的公司的etid

2. 更新线上数据库lz_datastore里面表et_info的etwebsite字段（官网url）

3. 更新线上数据库lz_datastore里面表et_contact的ecemail字段（email）

4. 钉钉信息推送
